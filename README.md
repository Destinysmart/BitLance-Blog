# BitLance Blog

The official blog for **BitLance**, the Bitcoin-native freelancing marketplace.

This repository powers **blog.bitlance.work**, where we publish educational content, industry insights, product updates, and practical guides for freelancers, employers, and builders participating in the Bitcoin economy.

---

## 🌍 About BitLance

BitLance connects talented professionals with companies that pay in Bitcoin.

Our mission is to build the leading marketplace for Bitcoin-native work by helping businesses hire vetted freelancers and enabling professionals worldwide to earn Bitcoin through meaningful remote opportunities.

Learn more at:

- Website: https://bitlance.work
- Blog: https://blog.bitlance.work *(coming soon)*

## 🚀 Local Development

Clone the repository:

```bash
git clone https://github.com/bitlance1/Blog.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
